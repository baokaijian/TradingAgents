平台架构

当前版本是面向 research-only 和 paper-trading 的 MVP。它不绕过风控，也不允许 LLM 直接连接券商接口；所有订单都必须先通过市场规则与账户风控。

01
TradingAgents 多 Agent 生成研究报告、交易计划与组合经理决策。
02
SignalNormalizer 把 Markdown 决策转换为结构化 SignalIntent。
03
MarketRuleEngine 检查 A 股/港股时段、手数、tick、涨跌幅与可卖数量。
04
RiskEngine 检查现金、单笔金额、仓位权重、置信度和黑名单。
05
PaperBroker 在仿真账户中生成订单状态、成交和持仓变化。

已落地模块

平台层代码位于 trading_platform/，与原始 tradingagents/ 研究框架分层放置，后续可独立扩展数据源、回测和券商网关。

A/H 证券解析

支持 600519.SH、000001.SZ、0700.HK，补齐市场、币种、手数、板块与涨跌幅属性。

市场规则检查

A 股支持 100 股手数、0.01 tick、涨跌幅和 T+1 可卖检查；港股支持每手股数、常用 tick table 与交易时段检查。

确定性风控

默认禁用市价单，检查最小置信度、最大单笔金额、最大仓位权重、可用现金与可卖持仓。

快速开始

使用 Python 3.10+ 安装并运行平台层测试。

git clone git@github.com:baokaijian/TradingAgents.git
cd TradingAgents
python3.12 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
python -m pytest tests/trading_platform -q
安全边界： 当前平台默认不开实盘、默认禁止市价单、默认 Hold 不生成订单、默认任何规则或风控失败都会拒绝订单。 实盘能力必须通过独立 broker gateway 显式接入，并保留完整审计。
